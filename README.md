# IITP 미세먼지 사회문제 해결 R&D

### 개발환경
- Vue 2
- SCSS
- LeafLet
- ECharts.js
- GraphQL
- Spring Boot
- JPA (Hibernate)
- QueryDSL
- PostgreSQL
- Docker

### 프로젝트 구조
|구분|Endpoint|설명|
|---|---|---|
|WEB|http://localhost:8085|웹 페이지|
|RESTful|http://localhost:9200/api|REST API|
|GraphQL|http://localhost:9200/graphql|GraphQL|
|Swagger|http://localhost:9200/swagger-ui.html|인터페이스 정의서|
